# Logs Directory

This directory stores security event logs generated by the Prompt Injection Defense System.

The file **security.log** records:

- Timestamp of the event
- Original user prompt
- Detected threat type
- Action taken by the system

Log files are generated automatically when the application runs.

For security and repository cleanliness, log files such as **security.log** are ignored from GitHub using the .gitignore file.
